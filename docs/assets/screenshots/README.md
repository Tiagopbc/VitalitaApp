- Em navegadores embutidos ou com storage particionado, o login Google do Firebase Auth pode falhar ao retornar pelo helper `__/auth/handler`. Para capturar telas internas, prefira uma conta demo com e-mail/senha ou um navegador normal ja autenticado.
